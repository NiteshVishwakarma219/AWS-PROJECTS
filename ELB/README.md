# ☁️ AWS Elastic Load Balancer (ELB) Project

## 🚀 Overview
This project demonstrates how AWS Elastic Load Balancer distributes traffic across multiple EC2 instances to ensure high availability and fault tolerance.

## 🧠 What I Learned
- AWS ELB (Application Load Balancer)
- EC2 instance deployment
- Target groups & health checks
- High availability architecture
- Traffic distribution concept

## 🏗️ Architecture
User → ELB → Multiple EC2 Instances (Multi-AZ)

## ⚙️ Services Used
- AWS EC2
- AWS Elastic Load Balancer
- Security Groups
- Target Groups

## 📸 Proof
Screenshots available in `/screenshots` folder

## 🔥 Key Feature
If one server fails, traffic automatically shifts to healthy server.
